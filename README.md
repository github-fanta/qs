# qs
B/S版本零件管理
实现技术： 
jquery+bootstrap+tiny框架([https://github.com/github-fanta/tiny-framework](https://github.com/github-fanta/tiny-framework))+sqlite3

# （一）功能包括：

  ## 1 用户部分：

    -用户登录;
    -修改密码;

  ## 2 零件管理：

    -零件的增、删、改、查（查询字段包括零件号和分零件）；
    -零件的出库、入库操作;
    -零件库存不足提醒（零件名变红）;
